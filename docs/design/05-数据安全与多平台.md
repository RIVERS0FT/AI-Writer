## 十四、本地数据库

推荐 SQLite。

Tauri 官方 SQL 插件支持 SQLite，并支持数据库迁移；迁移在事务中执行，失败时会回滚。([v2.tauri.app](https://v2.tauri.app/plugin/sql/))

核心表建议如下：

```text id="tdqenz"
projects
volumes
chapters
chapter_versions
outline_nodes

characters
character_states
relationships

world_entries
locations
items
organizations

timeline_events
foreshadowings

provider_configs
model_profiles
prompt_templates

generation_jobs
generation_steps
generation_outputs
usage_records

app_settings
schema_migrations
```

重要关系：

```text id="rrinsp"
project
 ├── volumes
 │    └── chapters
 │         └── chapter_versions
 ├── characters
 │    └── character_states
 ├── world_entries
 ├── timeline_events
 └── foreshadowings
```

`chapter_versions` 必须独立存在。

每次以下操作前创建版本：

- AI 重写
- 批量替换
- 导入覆盖
- 恢复旧稿
- 用户主动保存版本

## 十五、API Key 安全

API Key 不允许存入：

```text id="m0943k"
localStorage
明文 SQLite
普通 JSON 配置
Git 仓库
日志
错误堆栈
React 持久化状态
```

推荐使用 Tauri Stronghold。

Stronghold 插件面向机密信息和密钥保存，并提供 JavaScript 调用接口。([v2.tauri.app](https://v2.tauri.app/plugin/stronghold/))

存储流程：

```text id="yv297e"
用户输入 API Key
        ↓
React 立即提交给 Rust
        ↓
Rust 写入 Stronghold
        ↓
返回 apiKeyRef
        ↓
SQLite 只保存 apiKeyRef
```

请求模型时：

```text id="bs5pcm"
Rust 根据 apiKeyRef 读取密钥
        ↓
直接构造 HTTP 请求
        ↓
密钥不返回 React
```

还应做到：

- 日志自动遮盖 API Key
- 设置页只显示部分字符
- 导出项目时不导出密钥
- 删除 Provider 时同步删除密钥
- 剪贴板复制密钥后可选自动清除

## 十六、Tauri 权限安全

Tauri 2 使用 Capabilities 和 Permissions 控制前端能够调用哪些原生能力。([v2.tauri.app](https://v2.tauri.app/security/capabilities/))

不要给所有窗口开放全部能力。

示例：

```text id="4nbmzw"
主编辑窗口：
数据库读取
数据库写入
模型生成
文件导出

设置窗口：
Provider 配置
安全存储
应用设置

预览窗口：
只读项目内容
禁止安全存储
禁止文件写入
```

还应配置：

- 严格 CSP
- 禁止加载不受信任远程脚本
- 限制文件系统访问目录
- 限制 HTTP 请求域名
- 对所有 Command 参数进行校验
- 不在 WebView 中暴露通用 Shell 执行能力

## 十七、可选 Python 后端

Python 不是基础架构的必需部分。

标准版本：

```text id="bot58z"
React + TypeScript + Rust
```

已经足够完成：

- 模型 API 调用
- 流式生成
- Prompt 编排
- SQLite
- 全文检索
- 章节导出
- 一致性规则
- Token 预算

Python 适合用于：

- 本地 Transformers 模型
- PyTorch 推理
- 中文 NLP
- 高级文本评分
- 本地 Embedding
- 已有 Python 算法

桌面端可以将 Python 打包成 Sidecar。Tauri 官方支持嵌入外部二进制，并明确列举了由 PyInstaller 打包的 Python CLI 或 API 服务。([v2.tauri.app](https://v2.tauri.app/develop/sidecar/))

但不要让 Python Sidecar 成为 Android 和 iOS 必须依赖的核心。

推荐策略：

```text id="8vzbdx"
基础功能：
React + Rust，全平台运行

高级桌面功能：
可选 Python Sidecar

云端高级功能：
Python FastAPI 服务
```

## 十八、Web 版本

Web 版本复用：

```text id="trkirs"
React 页面
Tiptap 编辑器
TypeScript Core
Provider 接口
Prompt 模板
Zod Schema
```

替换平台能力：

| 原生版本 | Web 版本 |
|---|---|
| SQLite | IndexedDB |
| Stronghold | 会话密钥或加密存储 |
| 原生文件系统 | File System Access API / 下载 |
| Tauri HTTP | 浏览器 fetch 或服务端代理 |
| Tauri 通知 | Web Notification |
| 原生更新 | Service Worker |

Web 端最大限制是用户配置的模型服务可能不允许浏览器跨域访问。

因此 Web 版本建议支持两种模式：

```text id="w7myb5"
直接连接：
适用于允许浏览器跨域的模型服务

代理连接：
AI-Writer Server 转发模型请求
```

代理服务必须避免记录用户 API Key，并需要明确的安全设计。
